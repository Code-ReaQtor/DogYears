# DogYears

Convert human years to dog years and vice versa.


## Install

```bash
pip install dogyear
```

## Usage

```python
import dogyear

x = dogyear.get_dog_years(<age_in_human_years>, <size_in_pounds>)
y = dogyear.get_human_years(<age_in_dog_years>, <size_in_pounds>)
```
